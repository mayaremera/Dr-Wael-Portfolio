# Image assets

Place real photos here and update paths in `src/data/content.js`:

| File | Usage |
|------|--------|
| `dr-wael.jpg` | Main portrait (Profile section) |
| `hero-banner.jpg` | Top banner |
| `certificates/` | Scanned global certificates |
| `gallery/` | Conference and event photos |

Example after adding files:

```js
drWael: '/images/dr-wael.jpg',
```
