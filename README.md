# Dr. Wael Al-Dakroury — Portfolio Mockup

Elegant personal brand portfolio mockup for Dr. Wael A. Al-Dakroury, Speech-Language Pathologist.

Built with **React** (Vite) and **Tailwind CSS v4**.

## Design

- White background with refined teal accent (`#1a4d5c`)
- Cormorant Garamond (headings) + Source Sans 3 (body)
- Minimal animation, professional medical portfolio aesthetic
- Inspired by clean layouts from reference sites (ahmedgeneid.com, doniafahim.com)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for production

```bash
npm run build
npm run preview
```

## Project structure

```
src/
  components/
    Header.jsx      — Navigation
    Hero.jsx        — Introduction & stats
    About.jsx       — Biography
    Services.jsx    — Screening, counseling, assessment, treatment
    Expertise.jsx   — Why families choose Dr. Wael
    Leadership.jsx  — ASHA Fellow, roles, clinical focus
    Contact.jsx     — Contact form (UI mockup)
    Footer.jsx
  App.jsx
  index.css         — Tailwind + custom theme
```

Replace the portrait placeholder in `Hero.jsx` with a professional photo when available.
